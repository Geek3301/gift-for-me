# WordPress E-Commerce Portfolio Project

This repository contains the source code for a fully functional e-commerce website built with WordPress. The project is part of my portfolio and showcases my **full-stack WordPress development skills**.  

## Project Features
- Fully functional online store
- WooCommerce integration
- Custom theme and plugins
- Responsive design
- SEO optimization and basic security
- Product, order, and user management via admin panel

## Technologies
- WordPress
- PHP
- HTML, CSS, JavaScript
- MySQL
- WooCommerce
